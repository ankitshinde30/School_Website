# 🎓 School Website – Saraswati Vidya Mandir

A responsive **School Website** designed to provide information about the school, courses, facilities, and contact details.  

*A clean, responsive, and modern static school website built with HTML, CSS, and Bootstrap.*

This project represents **Saraswati Vidya Mandir** — a place dedicated to  
**Nurturing Knowledge • Building Character • Shaping Future**

---

# 🌐 Live Demo

🔗 https://school-website-one-beta.vercel.app/

*(Deployed on Vercel – feel free to explore!)*

---

# ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Modern Bootstrap 5 layout
- Attractive hero section with welcome message
- Background video support
- Multiple well-organized pages:
  - Home
  - About Us
  - Academics
  - Admissions
  - Events
  - Faculty
  - Gallery
  - Notice Board
  - Results
  - Achievements
  - Contact Us
- Student testimonials section
- Clean image & video asset management
- Simple and professional UI suitable for educational institutions

---

# 🚀 How to Run Locally

```bash

```

### 2️⃣ Open the project folder

```bash
cd School_Website
```

### 3️⃣ Run the website

Open **index.html** in your browser.
